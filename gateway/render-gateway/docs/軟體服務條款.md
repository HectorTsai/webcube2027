### 2. 使用限制
使用者承諾不進行以下禁止行為：
*   **逆向工程**：不得對本軟體進行還原編譯、反向工程、解編或嘗試獲取原始碼。
*   **散布租賃**：不得出租、租賃、出借、出售或以任何方式將本軟體再授權予第三人。
*   **修改繞過**：不得修改本軟體或繞過任何技術限制（如數位權利管理 DRM）。
*   **AI 抓取禁令 (2026 新增)**：嚴禁利用本軟體之輸出內容或內部架構進行人工智慧 (AI) 模型訓練。
