# 隱私權政策

本網站（以下簡稱「我們」）非常重視您的個人隱私。為了讓您安心使用本網站提供的各項服務，特此說明本網站的隱私權保護政策（以下簡稱「本政策」），以保障您的權益。

---

### 1. 個人資料的蒐集目的
我們蒐集您的個人資料，主要用於以下目的：
* 提供並優化網站服務（如帳號管理、內容推薦）。
* 客戶服務、爭議處理及身分驗證。
* 行銷業務、電子報發送（在取得您同意的前提下）。
* 履行法律義務或配合執法機關調查。

### 2. 蒐集之資料類別
當您使用本網站時，我們可能蒐集以下資料：
* **辨識個人者**：姓名、通訊地址、電話、電子郵件等。
* **技術資料**：IP 位址、瀏覽器類型、使用時間、操作路徑及 Cookie 紀錄。
* **AI 互動紀錄**：若您使用本站 AI 服務，系統可能會記錄對話過程以改進模型效能（但不包含個人敏感資訊）。

### 3. 個人資料利用之期間與地區
* **期間**：除法律另有規定外，資料將保留至您要求刪除帳號或服務結束後的一年內。
* **地區**：個人資料將用於本公司營運地區，以及提供雲端服務之伺服器所在地（如 AWS、Google Cloud 節點）。

### 4. 資料安全與保護
我們採用符合 2026 年標準的加密技術（如 TLS/SSL）保護傳輸中的資料，並設有嚴格的存取控制與資訊安全監控系統，防止資料遭未經授權的存取、洩漏或毀損。

### 5. 與第三人共用個資之政策
我們絕不會將您的個人資料提供給其他個人、團體或私人企業，但有下列情形者除外：
* 經由您的書面同意。
* 法律明文規定。
* 為維護國家安全或增進公共利益。
* 為免除您生命、身體、自由或財產上之危險。

### 6. Cookie 之使用
本網站會在您的瀏覽器中寫入並讀取 Cookie，以提供個人化功能。您可以透過瀏覽器設定拒絕 Cookie 的寫入，但這可能導致網站部分功能無法正常執行。

### 7. 使用者權利
根據個資法，您可以針對您的個人資料行使以下權利：
1. 查詢或請求閱覽。
2. 請求製給複製本。
3. 請求補充或更正。
4. 請求停止蒐集、處理或利用。
5. **請求刪除（被遺忘權）**。

如欲行使上述權利，請透過 [客服信箱或聯繫頁面連結] 與我們聯繫。

### 8. 政策修訂
本隱私權政策將因應法規變化與技術發展不定期修訂。修訂後之內容將公佈於本頁面，並於公告日起生效。
