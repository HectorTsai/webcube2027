# @dui/framework 規格書

> 最後更新：2026-07-22

---

## 1. 套件概述

`@dui/framework` 是 WebCube2027 管理型 Gateway 的**共用框架套件**。提供 Gateway 啟動引導（`createGateway()`）、HTTP 框架（Hono）、檔案路由載入器（`loadRoutes()`），並統一管理 CSS 框架（TailwindCSS v4 + daisyUI v5）的依賴版本。

管理型 Gateway（ai、page、cube、theme、data、auth）共用此套件，確保依賴版本一致，不需各自管理。

---

## 2. 職責範圍

### 2.1 在範圍內

- Gateway 啟動引導（`createGateway()`）
- 檔案路由系統（`loadRoutes()`）
- 統一匯出 Hono 核心
- 管理 TailwindCSS 與 daisyUI 的依賴版本

### 2.2 不在範圍內

- 不包含業務邏輯
- 不包含路由定義（路由由各 Gateway 自行定義）
- 不包含樣式配置（tailwind.config、daisyUI 主題等由各 Gateway 自行配置）

## 3. 匯出項目

### 3.1 `mod.ts` — 主要入口

```typescript
export { createGateway } from '@dui/framework';
export type { CreateGatewayOptions, Gateway } from '@dui/framework';
export { Hono } from 'hono';
```

`createGateway()` 是 Gateway 的統一啟動入口，處理以下初始化：

1. **L1 KV Store** — 在 `{dirname}/data/l1.json` 建立設定儲存
2. **加密金鑰** — 自動產生 256-bit 金鑰，存入 L1 的 `_crypto_key`，透過 `registerKey()` 註冊
3. **Hono 應用程式** — 建立 HTTP server
4. **檔案路由** — 自動載入 `{dirname}/routes/` 目錄下的 handler

| 匯出名稱 | 來源 | 型別 | 說明 |
|---------|------|------|------|
| `createGateway` | 本地 | function | Gateway 啟動引導，回傳 `Gateway` 物件 |
| `Gateway` | 本地 | interface | `{ app, l1, dataDir, port, start() }` |
| `CreateGatewayOptions` | 本地 | interface | `{ name, port?, dirname }` |
| `Hono` | `hono` | class | Web 框架核心類別 |

### 3.2 使用方式

```typescript
import { createGateway, Hono } from '@dui/framework';

const gw = await createGateway({
  name: 'my-gateway',
  port: 8000,
  dirname: import.meta.dirname!,  // ← 傳入 Gateway 自身目錄
});

// gw.app   — Hono 實例（已載入檔案路由）
// gw.l1    — L1Store（已初始化，加密金鑰已註冊）
// gw.dataDir — data/ 目錄絕對路徑
// gw.port  — HTTP 監聽埠

gw.start();
```

### 3.3 `loadRoutes()` — 檔案路由系統

```typescript
import { loadRoutes } from '@dui/framework/route-loader';
```

掃描 `routes/` 目錄，自動載入 handler 與靜態檔案。

**支援的檔案類型：**

| 類型 | 檔名 | 行為 |
|------|------|------|
| Method handler | `get.ts`, `post.ts`, `put.ts`, `del.ts`, `patch.ts` | 匯出對應 method 的 handler 函數 |
| 目錄首頁 | `index.tsx` | 視為 GET handler |
| 靜態檔案 | `.css`, `.js`, `.svg`, `.jpg`, `.png`, `.webp` | 自動以 GET 服務，含正確 Content-Type |
| 中間件 | `_middleware.ts` | 匯出 `middleware`（或 `default`）函數，套用至該目錄所有路由 |

**目錄命名規則：**
- `foo/` → 靜態路徑段落 `/foo`
- `_name_/` → 動態參數 `/:name`

**支援的 middleware：**

```typescript
import type { MiddlewareFn } from '@dui/framework/route-loader';

// middleware 可為 sync 或 async，回傳 Response 則中斷鏈路
const middleware: MiddlewareFn = async (c, next) => {
  // 到達 handler 前
  await next();
  // handler 回傳後
};
```

## 4. 管理依賴

| 依賴 | 版本 | 用途 | 消費方式 |
|------|------|------|---------|
| `hono` | ^4.12.31 | Web 框架 | workspace 根目錄 `imports` 全域共用 |
| `tailwindcss` | ^4.3.3 | CSS 框架 | 此套件管理版本，各 Gateway 自行執行 CLI |
| `@tailwindcss/cli` | ^4.3.3 | TailwindCSS CLI | 同上 |
| `daisyui` | ^5.6.18 | UI 元件庫 | 此套件管理版本 |
