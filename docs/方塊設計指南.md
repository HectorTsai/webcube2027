# 方塊設計指南（AI 提示詞用）

> 本文件僅供 AI 閱讀，生成方塊 JSON 時務必遵守。

## 1. 方塊定義 JSON 欄位

```typescript
{
  from: string;          // 必填："div" | "span" | "方塊:方塊:容器" | ...
  className?: string;    // 預設 CSS class（UnoCSS 原子類別）
  style?: Record<string, string>;
  args?: Record<string, ArgDef>;   // 參數定義：{ type, default, options }
  alpine?: { init?, bind?, model? };  // Alpine.js
  on?: Record<string, string>;     // { "click": "...", ... } → x-on:click="..."
  data?: Record<string, string>;   // { key: "value" } → data-key="value"
  slots?: Record<string, { from, children?, ... }>;  // 具名插槽
  children?: (object | string)[];  // 子方塊或純文字
  mergedArgs?: Record<string, string>;  // "@api/v1/..." 注入共享變數
}

interface ArgDef {
  type: "string" | "number" | "boolean";
  description?: string;
  default?: unknown;
  options?: string[];  // 字串類型列舉值
}
```

## 2. from 的兩種值

| from | 行為 |
|------|------|
| `"div"` `"span"` `"input"` ... | 原生 HTML 標籤 |
| `"方塊:方塊:容器"` 等 | 內建方塊元件（查 fallback registry） |

## 3. children 規則

- 陣列元素可為**物件**（子方塊定義）或**字串**（純文字）
- 遞迴上限 10 層
- **不要**用 `"text"` 欄位 → 改用 `children: ["文字"]`
- **不要**用 arbitrary CSS values（如 `text-[#ff0000]`）→ 用 UnoCSS preset 內的 class

### repeat 陣列展開

```json
{
  "repeat": "{items}",
  "from": "方塊:方塊:超連結",
  "href": "/page/{item}"
}
```

從 `mergedArgs.items` 取出陣列，每個元素建立一個方塊實例，元素值自動設為 `{item}`。

### $api 動態資料

```json
// 簡寫（預設查 /api/v1/info）
{ "$api": "商標" }

// 完整格式（自訂路徑 + 支援 {item} 插值）
{ "$api": { "path": "/api/v1/page/{item}", "field": "標題" } }
```

### mergedArgs 共享變數

```json
"mergedArgs": {
  "logo": "@api/v1/info/商標",
  "name": "@api/v1/info/名稱"
}
// 子方塊可用 {logo} {name} 引用
```

### slots.children 預設值

```json
"slots": {
  "header": {
    "from": "div",
    "children": [{ "from": "方塊:方塊:主選單", ... }]
  }
}
// children 為 slot 預設內容；外部 <Slot name="header"> 傳入時覆蓋
```

## 4. 內建方塊

可用方塊清單請動態查詢 `GET /api/v1/cube`，每個方塊的 `id` 即為 `from` 值。常見內建方塊範例：

```
方塊:方塊:容器      → 基礎容器（color/padding/gap/direction）
方塊:方塊:卡片      → 卡片（media + body slot）
方塊:方塊:主選單    → 頂部導航列（brand/content/drawer/footer slot）
方塊:方塊:頁尾      → 頁尾
方塊:方塊:基礎佈局  → 滿版佈局（header/content/footer slot）
方塊:方塊:抽屜      → 側邊滑入面板（backdrop/header/content/footer slot）
方塊:方塊:超連結    → 超連結
方塊:方塊:按鈕      → 按鈕
方塊:方塊:圖示      → 內建 SVG 圖示
方塊:方塊:圖片      → 圖片
方塊:方塊:列表      → 列表
```
