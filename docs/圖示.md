# 圖示與載入器系統設計

## 概述

WebCube2027 的圖示與載入器系統採用模組化設計，支援檔案系統和資料庫兩種載入方式，提供靈活且高效的使用體驗。

## 設計原則

### 1. 元件純粹性
- **不直接依賴資料庫**：元件透過 props 接收資料，保持純粹性
- **職責分離**：資料庫邏輯在路由層處理，元件只負責渲染
- **易於測試**：可以輕易傳入不同資料進行測試

### 2. 效能優先
- **直接 SVG 渲染**：使用 `dangerouslySetInnerHTML` 避免 React 包裝開銷
- **按需載入**：支援動態載入，不增加初始包大小
- **快取機制**：資料庫查詢結果快取，避免重複請求
## 載入器元件

### 設計決策

```typescript
// src/components/Loading.tsx
interface LoadingProps {
  風格?: 'spinner' | 'dots' | 'pulse' | 'skeleton'
  內容?: string
}

export default async function Loading({ 
  風格 = 'spinner', 
  內容 
}: LoadingProps) {
  try {
    // 1. 如果有傳入自定義 SVG 內容，直接使用
    if (內容 && 內容.includes('<svg')) {
      return <span dangerouslySetInnerHTML={{ __html: 內容 }} />
    }
    
    // 2. 否則嘗試從檔案載入預設風格
    const svgPath = new URL(`./loadings/${風格}.svg`, import.meta.url)
    const svgContent = await Deno.readTextFile(svgPath)
    
    return <span dangerouslySetInnerHTML={{ __html: svgContent }} />
  } catch (_e) {
    // 3. 後備方案：CSS 動畫
    return (
      <span className="inline-block animate-spin">
        <div className="w-4 h-4 border-2 border-slate-700 rounded-full border-t-transparent"></div>
      </span>
    )
  }
}
```

### 為什麼選擇這個設計？
#### 1. 直接 SVG 而非 React 元件
- **效能更好**：避免 React 包裝開銷
- **檔案更小**：不需要額外的 TypeScript 程式碼
- **維護簡單**：只需要維護 SVG 檔案

#### 2. `<span>` 而非 `<div>`
- **inline 佈局**：不會強制換行，可以和文字並排
- **靈活使用**：使用端完全控制佈局
- **語意正確**：載入器通常是內聯元素

#### 3. 三層後備機制
- **自定義 SVG**：最大靈活性
- **檔案系統**：預設風格，快速載入
- **CSS 動畫**：最終保險，確保總有載入器

### 使用場景

```typescript
// 1. 預設風格
<Loading 風格="spinner" />
<Loading 風格="dots" />
<Loading 風格="pulse" />
<Loading 風格="skeleton" />

// 2. 自定義 SVG
<Loading 內容="<svg class='w-4 h-4'>...</svg>" />

// 3. 與文字並排
<span>載入中...<Loading 風格="spinner" /></span>
<span>處理中<Loading 風格="dots" /></span>
<span>上傳中<Loading 內容="<svg class='w-4 h-4'>...</svg>" /></span>
```

## 最佳實踐

### 1. 新增載入器
1. 在 `src/components/loadings/` 加入 SVG 檔案
2. 更新 `LoadingProps` 的 union type
3. 更新文件說明

### 2. 資料庫整合
1. 在 `圖示.json` 加入資料
2. 使用 `圖示服務` 載入
3. 在路由層傳遞給元件

### 3. 使用方式
```typescript
// 預設風格
<Loading 風格="spinner" />

// 自定義內容
<Loading 內容="<svg>...</svg>" />

// 與文字並排

```
