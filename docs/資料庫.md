# 資料庫架構設計

## 概述
WebCube2027 採用混合資料庫架構：KV 存儲系統資料，SurrealDB 存儲多租戶資料。

## 資料庫分層

### 系統資料庫 (KV)
- **用途**：儲存系統配置、快取資料
- **特點**：高效能、簡單鍵值對
- **內容**：系統設定、連線資訊、快取
- **實作**：`database/kv.ts` - Deno KV 封裝

### 共享資料庫 (SurrealDB - webcube namespace)
- **用途**：儲存跨租戶的共享資料
- **內容**：網站資訊、佈景主題、骨架、配色、系統單字、圖示
- **隔離**：使用 namespace 區隔
- **實作**：`database/surrealdb.ts` - SurrealDB HTTP API 封裝

### 租戶資料庫 (SurrealDB - 獨立實例)
- **用途**：各網站的專用資料
- **內容**：內容、頁面、單字、標籤、媒體、圖示
- **隔離**：每個 domain 獨立實例
- **實作**：動態建立連線，根據 domain 區隔

## KV 資料庫實作

### KV資料庫類
```typescript
// database/kv.ts
export class KV資料庫 {
  private kv!: Deno.Kv

  async 開啟(): Promise<void> {
    this.kv = await Deno.openKv("data/kv.sqlite3")
  }

  async 取得資料<T extends 資料>(id: string): Promise<T | unknown | null>
  async 寫入資料(model: 資料): Promise<void>

  async 初始化(model: string): Promise<void>

```

### 特點
- **動態模型載入**：根據 ID 中的 type 自動載入對應模型
- **種子資料初始化**：表為空時自動載入 seeds
- **簡單鍵值對**：直接使用 Deno KV API

## 多租戶隔離

### 識別機制
- **網站 ID**：`domain` 為站點唯一識別符
- **連線管理**：根據 domain 動態建立資料庫連線
- **權限控制**：以 domain 控制資料存取範圍

### 連線策略
```typescript
// 系統資料庫 - 固定連線
const systemKV = new KV資料庫()
await systemKV.開啟()

// 共享資料庫 - 固定連線，webcube namespace
const sharedDB = new Surreal資料庫({
  url: 'ws://localhost:8000',
  namespace: 'webcube',
  database: 'webcube',
  user: 'root',
  password: 'root'
})

// 租戶資料庫 - 動態連線
function getTenantDB(domain: string) {
  return new Surreal資料庫({
    url: 'ws://localhost:8000',
    namespace: domain,
    database: domain,
    user: 'tenant_user',
    password: 'tenant_password'
  })
}
```

### 資料隔離實作
```typescript
// Middleware 中的資料庫連線管理
app.use('*', async (ctx, next) => {
  const domain = ctx.req.host
  
  // 設定資料庫連線到 Context
  ctx.set('systemKV', systemKV)
  ctx.set('sharedDB', sharedDB)
  ctx.set('tenantDB', getTenantDB(domain))
  
  await next()
})
```

### 自動載入機制
```typescript
// 讀取種子資料
export async function 讀取種子<T extends 資料>(model: string): Promise<T[] | null> {
  try {
    const seedPath = new URL(`./seeds/${model}.json`, import.meta.url)
    const text = await Deno.readTextFile(seedPath)
    const raw = JSON.parse(text) as Record<string, unknown>[]
    
    const mod = await import(`./models/${model}.ts`)
    const Model = mod.default
    
    return raw.map((item) => new Model(item))
  } catch (_err) {
    return null
  }
}
```
