# 動畫使用書

WebCube 動畫系統提供兩種動畫方式：**組合式動畫**（可疊加多種效果）和 **預設動畫**（Animate.css 風格）。

---

## 🎯 快速開始

### 組合式動畫（推薦）

組合式動畫可以**同時疊加多種效果**，例如同時淡入、滑動和縮放：

```html
<div class="animate-in fade-in slide-in-from-bottom zoom-in duration-500">
  內容會淡入 + 從下方滑入 + 縮放進入
</div>
```

### 預設動畫

直接使用 Animate.css 風格的動畫類名：

```html
<div class="animate-bounce-in">彈跳進入效果</div>
```

## 📚 組合式動畫類名

### 基礎類

| 類名 | 說明 |
|------|------|
| `animate-in` | 進入動畫基礎類（必須） |
| `animate-out` | 退出動畫基礎類（必須） |

### 淡入淡出

| 類名 | 效果 |
|------|------|
| `fade-in` | 淡入（從透明到不透明） |
| `fade-out` | 淡出（從不透明到透明） |
| `fade-in-50` | 淡入到 50% 透明度 |
| `fade-out-75` | 淡出到 75% 透明度 |

**範例：**
```html
<div class="animate-in fade-in duration-300">淡入效果</div>
```

### 縮放

| 類名 | 效果 |
|------|------|
| `zoom-in` | 從 0 縮放到原始大小 |
| `zoom-out` | 從原始大小縮放到 0 |
| `zoom-in-50` | 從 50% 縮放到原始大小 |
| `zoom-out-80` | 從原始大小縮放到 80% |

**範例：**
```html
<div class="animate-in zoom-in duration-500">縮放進入</div>
```

### 旋轉

#### Z 軸旋轉（2D）
| 類名 | 效果 |
|------|------|
| `spin-in` | 旋轉 360 度進入 |
| `spin-out` | 旋轉 360 度退出 |
| `spin-in-90` | 旋轉 90 度進入 |
| `spin-out-180` | 旋轉 180 度退出 |

#### X 軸旋轉（3D）
| 類名 | 效果 |
|------|------|
| `spin-in-x` | X 軸旋轉 90 度進入 |
| `spin-out-x` | X 軸旋轉 90 度退出 |
| `spin-in-x-45` | X 軸旋轉 45 度進入 |

#### Y 軸旋轉（3D）
| 類名 | 效果 |
|------|------|
| `spin-in-y` | Y 軸旋轉 90 度進入 |
| `spin-out-y` | Y 軸旋轉 90 度退出 |
| `spin-in-y-180` | Y 軸旋轉 180 度進入 |

**範例：**
```html
<div class="animate-in spin-in-y-180 duration-800">翻轉進入</div>
```

### 滑動

| 類名 | 效果 |
|------|------|
| `slide-in-from-top` | 從上方滑入 |
| `slide-in-from-bottom` | 從下方滑入 |
| `slide-in-from-left` | 從左側滑入 |
| `slide-in-from-right` | 從右側滑入 |
| `slide-in-from-top-50` | 從上方滑入 50% |

#### 退出方向
| 類名 | 效果 |
|------|------|
| `slide-out-to-top` | 向上方滑出 |
| `slide-out-to-bottom` | 向下方滑出 |
| `slide-out-to-left` | 向左側滑出 |
| `slide-out-to-right` | 向右側滑出 |

**範例：**
```html

```

### 模糊

| 類名 | 效果 |
|------|------|
| `blur-in` | 從模糊到清晰（8px） |
| `blur-out` | 從清晰到模糊（8px） |

**範例：**


| 類名 | 效果 |
|------|------|
| `origin-center` | 中心點（預設） |
| `origin-top` | 上方中心 |
| `origin-top-right` | 右上角 |

## 🎨 動畫參數控制

### 持續時間

| 類名 | 時長 |
|------|------|
| `duration-100` | 100ms |
| `duration-300` | 300ms |
| `duration-500` | 500ms |
| `duration-1000` | 1000ms (1秒) |

### 延遲

| 類名 | 延遲 |
|------|------|
| `delay-100` | 延遲 100ms |
| `delay-300` | 延遲 300ms |
| `delay-500` | 延遲 500ms |

### 重複次數

| 類名 | 效果 |
|------|------|
| `repeat-1` | 播放 1 次（預設） |
| `repeat-2` | 播放 2 次 |
| `repeat-3` | 播放 3 次 |
| `repeat-infinite` | 無限循環 |

### 填充模式

| 類名 | 效果 |
|------|------|
| `fill-none` | 動畫結束後不保持狀態 |
| `fill-forwards` | 保持動畫結束狀態 |
| `fill-backwards` | 保持動畫開始狀態 |
| `fill-both` | 同時保持開始和結束狀態 |

### 方向

| 類名 | 效果 |
|------|------|
| `direction-normal` | 正常播放 |
| `direction-reverse` | 反向播放 |
| `direction-alternate` | 來回播放 |
| `direction-alternate-reverse` | 反向來回播放 |

### 緩動函數

| 類名 | 效果 |
|------|------|
| `ease-linear` | 線性 |
| `ease-in` | 慢進快出 |
| `ease-out` | 快進慢出 |
| `ease-in-out` | 慢進慢出 |

**範例：**
```html
<div class="animate-in fade-in duration-1000 delay-200 ease-in-out fill-forwards">
  1秒淡入，延遲200ms，保持最終狀態
</div>
```

## 💡 使用範例

### 範例 1：卡片進入動畫
```html
<div class="animate-in fade-in slide-in-from-bottom zoom-in-95 duration-500 ease-out">
  <h2>標題</h2>
  <p>內容會同時淡入、從下滑入、並稍微縮放</p>
</div>
```

### 範例 2：按鈕點擊效果
```html
<button class="animate-tada">
  點我！（會有慶祝效果）
</button>
```

### 範例 3：通知彈出
```html
<div class="animate-in slide-in-from-right duration-300 fill-forwards">
  通知訊息從右側滑入並保持
</div>
```

### 範例 4：載入動畫
```html
<div class="animate-spin repeat-infinite">
  ⏳ 載入中...
</div>
```

### 範例 5：複雜組合
```html
<div class="animate-in fade-in zoom-in spin-in-y-180 slide-in-from-top duration-1000 delay-200 ease-in-out">
  同時具備：淡入 + 縮放 + Y軸旋轉 + 從上滑入
</div>
```

### 範例 6：退出動畫
```html
<div class="animate-out fade-out slide-out-to-left duration-400 fill-forwards">
  內容會淡出並向左滑出
</div>
```

### 範例 7：3D 翻轉卡片
```html
<div class="animate-in spin-in-y-180 perspective-600 origin-center duration-800 ease-out">
  卡片從 Y 軸翻轉進入，帶有 3D 透視效果
</div>
```

### 範例 8：傾斜進入效果
```html
<div class="animate-in fade-in skew-in-x-12 slide-in-from-left duration-500">
  同時淡入、傾斜、滑入
</div>
```

### 範例 9：水平拉伸動畫
```html
<div class="animate-in scale-x-in origin-left duration-600">
  從左側水平拉伸展開
</div>
```

### 範例 10：從角落旋轉
```html
<div class="animate-in spin-in-360 origin-top-left fade-in duration-700">
  從左上角旋轉進入並淡入
</div>
```

## 🎯 最佳實踐

1. **組合式動畫優先**：大部分情況使用組合式動畫，更靈活
2. **控制持續時間**：建議 300-500ms，避免過長影響體驗
3. **適當使用延遲**：多個元素可用 `delay-*` 製造錯落效果
4. **保持最終狀態**：進入動畫用 `fill-forwards` 保持結束狀態
5. **避免過度使用**：不是所有元素都需要動畫

---

## 📝 注意事項

- 組合式動畫必須搭配 `animate-in` 或 `animate-out` 基礎類
- 預設動畫可以直接使用，不需要基礎類
- 數字參數支援任意數值（如 `duration-350`、`fade-in-65`）
- 3D 旋轉效果在某些瀏覽器可能需要硬體加速
