# 多國語言支援設計

## 概述
WebCube2027 採用 @dui/smartmultilingual 實現多國語言支援，提供多層翻譯機制：資料庫翻譯 → AI 翻譯 → Google 翻譯，並支援翻譯結果回寫。

## 多層翻譯機制

### 翻譯優先順序
1. **網站資料庫單字查找** (第一優先)
2. **系統資料庫單字查找** (第二優先)  
3. **AI 翻譯** (可選，需網站設定啟用)
4. **Google 翻譯** (最後保險措施)

### 翻譯結果回寫
- **優先寫回網站資料庫**：如果網站資料庫存在
- **次優寫回系統資料庫**：如果無網站資料庫但系統資料庫存在
- **不寫回**：如果兩者都不存在

### 部署階段說明
1. **無系統資料庫**：整個專案剛部署，部署者尚未設定系統資料庫
2. **無網站資料庫**：網站申請者剛指向此系統，但尚未通過申請或設定

## @dui/smartmultilingual 整合

### 註冊自訂翻譯函數
```typescript
import { SmartMultilingual, GoogleTranslation } from '@dui/smartmultilingual'

class WebCube多語言管理器 {
  private ai: SmartMultilingual
  
  constructor() {
    this.ai = new SmartMultilingual()
    
    // 註冊多層翻譯機制
    this.ai.registerTranslator('webcube', this.多層翻譯.bind(this))
    
    // 註冊 Google 翻譯作為保險措施
    this.ai.registerTranslator('google', new GoogleTranslation())
  }
  
  private async 多層翻譯(
    text: string, 
    from: string, 
    to: string, 
    options: any
  ): Promise<string> {
    // 1. 網站資料庫查找
    const 網站翻譯 = await this.網站資料庫查找(text, from, to)
    if (網站翻譯) return 網站翻譯
    
    // 2. 系統資料庫查找
    const 系統翻譯 = await this.系統資料庫查找(text, from, to)
    if (系統翻譯) return 系統翻譯
    
    // 3. AI 翻譯 (可選)
    if (await this.檢查AI翻譯啟用()) {
      try {
        const ai翻譯 = await this.ai.translate(text, from, to)
        await this.寫回翻譯結果(text, from, to, ai翻譯)
        return ai翻譯
      } catch (error) {
        console.warn('[多國語言] AI翻譯失敗，降級為Google翻譯:', error)
      }
    }
    
    // 4. Google 翻譯 (保險措施)
    try {
      const google翻譯 = await this.ai.translate(text, from, to, { 
        translator: 'google' 
      })
      await this.寫回翻譯結果(text, from, to, google翻譯)
      return google翻譯
    } catch (error) {
      console.error('[多國語言] Google翻譯也失敗:', error)
      return text // 返回原文
    }
  }
}
```

## 語言處理優先順序

### 1. URL 路徑 (最高優先級)
```typescript
// /en/index → 語言 = en
// /zh-tw/about → 語言 = zh-tw
// /ja/contact → 語言 = ja

function 從URL提取語言(path: string): string | null {
  const match = path.match(/^\/([a-z]{2}(-[a-z]{2})?)\//)
  return match ? match[1] : null
}
```

### 2. Cookie 記憶 (第二優先級)
```typescript
// webcube-lang Cookie (1年過期)
const 語言Cookie = 'webcube-lang'

function 設定語言Cookie(ctx: Context, 語言: string): void {
  ctx.cookie(語言Cookie, 語言, {
    maxAge: 365 * 24 * 60 * 60, // 1年
    httpOnly: false,
    secure: true,
    sameSite: 'Lax'
  })
}

function 從Cookie取得語言(ctx: Context): string | null {
  return ctx.cookie(語言Cookie)
}
```

### 3. 瀏覽器設定 (第三優先級)
```typescript
// accept-language header
function 從瀏覽器取得語言(ctx: Context): string | null {
  const acceptLanguage = ctx.req.header('accept-language')
  if (!acceptLanguage) return null
  
  // 解析 accept-language: "zh-TW,zh;q=0.9,en;q=0.8,en-US;q=0.7"
  const 語言們 = acceptLanguage
    .split(',')
    .map(lang => lang.split(';')[0].trim().toLowerCase())
    .map(lang => lang.replace('-', '_'))
  
  return 語言們[0] || null
}
```

### 4. 預設語言 (最低優先級)
```typescript
const 預設語言 = 'zh-tw'
```

## 資料庫翻譯查找

### 網站資料庫查找
```typescript
class 網站翻譯服務 {
  constructor(private tenantDB: Surreal資料庫) {}
  
  async 查找單字(原文: string, 來源語言: string, 目標語言: string): Promise<string | null> {
    const 查詢 = `
      SELECT 資料.目標語言 
      FROM 單字 
      WHERE 資料.來源語言 = $原文 
      AND 資料.語言 = $目標語言
      LIMIT 1
    `
    
    const 結果 = await this.tenantDB.查詢(查詢, {
      原文,
      來源語言,
      目標語言
    })
    
    return 結果[0]?.目標語言 || null
  }
}
```

## 翻譯結果回寫

### 寫回策略
```typescript
class 翻譯回寫服務 {
  constructor(
    private tenantDB: Surreal資料庫 | null,
    private sharedDB: Surreal資料庫 | null
  ) {}
  
  async 寫回翻譯結果(
    原文: string, 
    來源語言: string, 
    目標語言: string, 
    翻譯結果: string
  ): Promise<void> {
    const 單字資料 = {
      資料: {
        [來源語言]: 原文,
        [目標語言]: 翻譯結果
      },
      語言: 目標語言
    }
    
    // 1. 優先寫回網站資料庫
    if (this.tenantDB) {
      try {
        await this.tenantDB.查詢(`
          CREATE 單字 CONTENT ${JSON.stringify(單字資料)}
        `)
        console.log(`[多國語言] 翻譯結果已寫回網站資料庫: ${原文} → ${翻譯結果}`)
        return
      } catch (error) {
        console.warn('[多國語言] 寫回網站資料庫失敗:', error)
      }
    }
    
    // 2. 次優寫回系統資料庫
    if (this.sharedDB) {
      try {
        await this.sharedDB.查詢(`
          CREATE 單字 CONTENT ${JSON.stringify(單字資料)}
        `)
        console.log(`[多國語言] 翻譯結果已寫回系統資料庫: ${原文} → ${翻譯結果}`)
        return
      } catch (error) {
        console.warn('[多國語言] 寫回系統資料庫失敗:', error)
      }
    }
    
    // 3. 都失敗則不寫回
    console.warn('[多國語言] 無法寫回翻譯結果: 無可用的資料庫')
  }
}
```

## AI 翻譯設定

### 網站 AI 翻譯設定
```typescript
// 在 網站資訊 模型中添加 AI 翻譯設定
export default class 網站資訊 extends 資料 {
  // ... 其他欄位
  
  public AI翻譯設定: {
    啟用: boolean
    提供商: 'openai' | 'ollama' | 'gemini'
    模型: string
    API金鑰: SecretString
    頻率限制: number // 每小時請求限制
  }
  
  constructor(data: Record<string, unknown> = {}, 權限設定: 權限 = { 讀: true, 寫: true, 刪除: true }) {
    super(data, 權限設定)
    
    this.AI翻譯設定 = {
      啟用: (data?.AI翻譯設定 as any)?.啟用 ?? false,
      提供商: (data?.AI翻譯設定 as any)?.提供商 ?? 'ollama',
      模型: (data?.AI翻譯設定 as any)?.模型 ?? 'llama2',
      API金鑰: new SecretString({ 
        cipherText: (data?.AI翻譯設定 as any)?.API金鑰 
      }),
      頻率限制: (data?.AI翻譯設定 as any)?.頻率限制 ?? 100
    }
  }
}
```

### AI 翻譯檢查
```typescript
async 檢查AI翻譯啟用(): Promise<boolean> {
  const 網站資訊 = await this.取得當前網站資訊()
  
  if (!網站資訊?.AI翻譯設定?.啟用) {
    return false
  }
  
  // 檢查頻率限制
  const 目前使用量 = await this.取得AI翻譯使用量()
  if (目前使用量 >= 網站資訊.AI翻譯設定.頻率限制) {
    console.warn('[多國語言] AI翻譯頻率限制已達')
    return false
  }
  
  return true
}
```

## Middleware 實作

### 語言檢測 Middleware
```typescript
export default async function 語言檢測Middleware(ctx: Context, next: Next) {
  // 1. 嘗試從 URL 提取
  let 語言 = 從URL提取語言(ctx.req.path)
  
  // 2. 嘗試從 Cookie 取得
  if (!語言) {
    語言 = 從Cookie取得語言(ctx)
  }
  
  // 3. 嘗試從瀏覽器設定取得
  if (!語言) {
    語言 = 從瀏覽器取得語言(ctx)
  }
  
  // 4. 使用預設語言
  if (!語言) {
    語言 = 預設語言
  }
  
  // 5. 驗證語言是否支援
  const 支援的語言們 = await 取得支援的語言們()
  if (!支援的語言們.includes(語言)) {
    語言 = 預設語言
  }
  
  // 6. 設定多語言管理器到 Context
  const 多語言管理器 = new WebCube多語言管理器(
    ctx.get('tenantDB'),
    ctx.get('sharedDB')
  )
  
  ctx.set('語言', 語言)
  ctx.set('多語言管理器', 多語言管理器)
  設定語言Cookie(ctx, 語言)
  
  console.log(`[語言檢測] ${ctx.req.path} → ${語言}`)
  
  await next()
}
```

## 多語言字串系統

### MultilingualString 增強
```typescript
class 增強MultilingualString extends MultilingualString {
  constructor(
    data: Record<string, string> | undefined,
    private 多語言管理器: WebCube多語言管理器,
    private 當前語言: string
  ) {
    super(data)
  }
  
  // 使用現有的 toStringAsync 方法
  async toStringAsync(語言?: string): Promise<string> {
    const 目標語言 = 語言 || this.當前語言
    
    // 1. 使用現有翻譯
    if (this.data[目標語言]) {
      return this.data[目標語言]
    }
    
    // 2. 找到原文
    const 原文 = this.找到原文()
    if (!原文) return ''
    
    // 3. 使用多層翻譯機制
    try {
      const 翻譯結果 = await this.多語言管理器.多層翻譯(
        原文, 
        'zh-tw', // 假設原文是繁體中文
        目標語言,
        {}
      )
      
      // 4. 快取翻譯結果
      this.data[目標語言] = 翻譯結果
      
      return 翻譯結果
    } catch (error) {
      console.error('[多國語言] 翻譯失敗:', error)
      return 原文
    }
  }
  
  private 找到原文(): string | null {
    // 優先順序：預設語言 > 英文 > 第一個可用語言
    if (this.data[預設語言]) return this.data[預設語言]
    if (this.data['en']) return this.data['en']
    
    const 可用語言們 = Object.keys(this.data)
    return 可用語言們.length > 0 ? this.data[可用語言們[0]] : null
  }
}
```

### 使用範例
```typescript
// 在元件中使用
export default function 首頁元件(ctx: Context) {
  const 多語言管理器 = ctx.get('多語言管理器')
  const 當前語言 = ctx.get('語言')
  
  const 歡迎訊息 = new 增強MultilingualString({
    'zh-tw': '歡迎來到 WebCube2027',
    'en': 'Welcome to WebCube2027'
  }, 多語言管理器, 當前語言)
  
  // 使用 toStringAsync 取得翻譯
  const 顯示文字 = await 歡迎訊息.toStringAsync()
  
  return (
    <div>
      <h1>{顯示文字}</h1>
    </div>
  )
}
```
