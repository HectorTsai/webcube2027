## 📖 文件使用指南

### Markdown 格式
所有文件都使用 Markdown 格式，支援：
- 程式碼高亮
- 表格和列表
- 連結和圖片
- 中文內容支援

### 程式碼範例
文件中的程式碼範例都使用 TypeScript，並遵循專案的中文命名慣例：

```typescript
class 網站資訊 {
  網域!: string
  名稱: MultilingualString
  建立時間: Date = new Date()
}
```

### 連結約定
- 內部連結使用相對路徑：`[路由器](./路由器.md)`
- 外部連結使用完整 URL：`[Deno](https://deno.land)`
- GitHub 連結使用簡短格式：`[#123](https://github.com/issue/123)`
