## 📋 專案概述

WebCube2027 是一個基於 Deno + Hono 的多站點網站生成器，支援 AI 驅動的佈景主題生成、檔案路由器、多租戶隔離等現代化功能。

## 🎯 核心目標

- **多站點支援**：以 domain 區分的獨立網站
- **AI 驅動**：自動生成佈景主題和配色
- **檔案路由**：自行設計的易抽離路由器
- **多租戶**：完整的資料隔離機制
- **現代化**：Deno + Hono + UnoCSS + SurrealDB

## 🏗 系統架構

### 技術棧
- **運行時**：Deno 2.x
- **框架**：Hono
- **資料庫**：KV (系統) + SurrealDB (多租戶)
- **媒體**：MinIO
- **AI**：本地 ollama + @dui/smartmultilingual
- **UI**：AI 生成佈景主題 + UnoCSS
- **部署**：Docker

### 多租戶隔離
- **網站識別**：`domain` 為站點 ID
- **系統資料**：KV 存儲系統配置
- **共享資料**：SurrealDB `webcube` namespace
- **租戶資料**：各網站獨立 SurrealDB 實例

## 🔄 Middleware 資料處理

### 語言處理優先順序
1. **URL 路徑** - `/en/index` → `en`
2. **Cookie 記憶** - `webcube-lang` Cookie (1年過期)
3. **瀏覽器設定** - `accept-language` header
4. **預設語言** - `zh-tw`

### State 設定
```typescript
ctx.set('骨架資料', 骨架資料);
ctx.set('配色資料', 配色資料);
ctx.set('佈景主題資料', 佈景主題資料);
ctx.set('語言', lang);
```

### **內部重導向機制** (✅ 已實作)
- **開發者主導**：提供 `ctx.internalRedirect()` 和 `ctx.redirect()`
- **高效能優先**：避免重新 request，減少資料庫查詢
- **安全閥機制**：循環時自動降級為外部重導向

## 📖 開發風格

### 中文命名支援
- 允許 class/變數/參數用中文
- 同時保留英文欄位以利資料交換

### 範例
```typescript
class 網站資訊 {
  網域!: string // domain
  名稱: MultilingualString
  描述: MultilingualString
  佈景主題: string // theme id
  配色: string // palette id
  骨架: string // layout id
  建立時間: Date = new Date()
}
```

### Log 格式
統一使用 `[filename] message` 格式，方便定位錯誤來源：
```typescript
console.error("[surrealdb.ts] SurrealDB query failed: 500")
```
